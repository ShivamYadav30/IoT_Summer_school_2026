#Q-12. MICROCONTROLLER vs MICROPROCESSOR

| **Parameter**         | **Microcontroller (Arduino UNO)**                                     | **Microprocessor (Raspberry Pi)**                                      |
| --------------------- | --------------------------------------------------------------------- | ---------------------------------------------------------------------- |
| **CPU Speed**         | **16 MHz** (ATmega328P)                                               | **1.2–2.4 GHz** (depends on Raspberry Pi model)                        |
| **On-chip RAM**       | **2 KB SRAM**                                                         | **No on-chip RAM** (uses external RAM, typically **1 GB–8 GB**)        |
| **On-chip Flash**     | **32 KB Flash Memory**                                                | **No on-chip Flash** (boots from **microSD card** or external storage) |
| **Operating System**  | **No Operating System** (runs a single program directly)              | **Linux-based OS** (e.g., Raspberry Pi OS, Ubuntu)                     |
| **Typical Use Case**  | Embedded systems, sensor interfacing, robotics, automation, IoT nodes | Image processing, AI/ML, multimedia, web servers, IoT gateways         |
| **Power Consumption** | **Low** (≈ 50–70 mA at 5 V, ~0.25–0.35 W)                             | **Higher** (≈ 2–7 W depending on model and workload)                   |
