Q-13: ABOUT ARDUINO UNO PINS AND REAL IoT USE OF EACH.




|**Pin Type**       | **Arduino UNO Pins**                            | **Explanation**                                                                                                      | **Example Device**                                   | **Real IoT Use Case**                                                                                                           |
| ------------------ | ----------------------------------------------- | -------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| **Digital Input**  | D0–D13                                          | Reads only two logic states: **HIGH (1)** or **LOW (0)**. Used to detect ON/OFF signals from sensors or switches.    | Push button, PIR motion sensor                       | **Smart Home Security:** A PIR sensor detects motion and sends a HIGH signal to trigger an IoT security alert.                  |
| **Digital Output** | D0–D13                                          | Sends HIGH (5V) or LOW (0V) signals to control external devices.                                                     | LED, Relay, Buzzer                                   | **Smart Lighting:** Arduino turns an LED or relay ON/OFF remotely through an IoT platform.                                      |
| **Analog Input**   | A0–A5                                           | Reads analog voltages (0–5V) using a **10-bit ADC**, producing values from **0–1023**.                               | Temperature sensor (LM35), LDR, Soil moisture sensor | **Smart Agriculture:** Reads soil moisture to automatically control irrigation.                                                 |
| **PWM Output**     | D3, D5, D6, D9, D10, D11                        | Generates **Pulse Width Modulation (PWM)** signals to simulate analog output. Used for speed and brightness control. | Servo motor, DC motor, LED                           | **Smart Fan Control:** Adjusts fan speed automatically based on room temperature.                                               |
| **I2C Pins**       | **A4 (SDA), A5 (SCL)**                          | Two-wire communication protocol for connecting multiple devices using only two data lines.                           | OLED display, RTC module, BME280 sensor              | **Weather Monitoring Station:** Reads temperature, humidity, and pressure from an I2C sensor and uploads the data to the cloud. |
| **SPI Pins**       | **D10 (SS), D11 (MOSI), D12 (MISO), D13 (SCK)** | High-speed serial communication protocol for connecting peripherals such as memory cards and displays.               | SD card module, RFID module                          | **IoT Data Logger:** Stores sensor readings on an SD card before transmitting them to a cloud server.                           |
